# OffWorkKit - 下班工具箱

## 📋 项目愿景

**致汝州一高的老师们：**

我们深知，您们的核心价值在于传道授业与启发思考，而非深陷于繁杂重复的事务性工作。OffWork
是专为我校教学与管理场景开发的数字助手，旨在通过贴心、高效的辅助工具，将老师们从耗时费力的重复性任务中解放出来，从而提升工作效率，
**增加宝贵的休息时间**，让教育者能更专注于教学本身与学生成长。

## ✨ 核心功能

OffWorkKit 集成了一系列针对教师日常工作场景的实用工具，操作简单，功能强大：

| 功能模块           | 核心痛点                                       | OffWork 如何辅助您                                                                                                                                        |
|:---------------|:-------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| **📚 自动刷课**    | 各类继续教育平台课程需保持在线，占用老师宝贵的业余时间。               | 自动完成视频播放、章节切换等流程，增加时间。                                                                                                                               |
| **📅 辅助排课**    | 手动排课需反复在表格中对照、填写，易出现**撞课**等，且调整困难。         | 1. **冲突实时预警**：在您安排课程时，系统自动检查，若出现撞课，会**立即高亮提示**，杜绝低级错误。<br>2. **便捷填表**：提供教师、科目、教室的快速选择面板，支持在课表上“点点点”完成填写，告别手动输入。<br>3. **联动津贴计算**：根据教师分工及课表，自动统计课时数   |
| **💰 津贴计算**    | 津贴计算需人工从课表中逐个统计每位老师的课时，且遇到考试、活动等情况时统计更加混乱。 | 1. **自动课时统计**：系统直接读取您排好的课表，**自动统计**任意日期区间内所有教师的实际课时数，生成清晰明细。<br>2. **智能日历联动**：内置日历功能。如遇考试、运动会等活动，您只需在日历上标记占用的课时，统计时便会**自动排除这些课时**，确保统计结果与实际教学工作完全匹配。 |
| **📈 成绩分析**    | 重复性工作繁重，耗时费力。                              | 快速导入成绩表，设置总分与各科分数线，系统**自动统计**出单双上线学生数量                                                                                                               |
| **🗂️ PDF工具箱** | 教学材料常需合并、分割或压缩，操作繁琐。                       | 集成轻量级的PDF处理工具，轻松完成合并、分割与压缩，方便材料整理与分发。                                                                                                                |

## 🚀 技术特色

- **🖥️ 轻量桌面应用**：无需浏览器，打开即用。基于现代技术开发，体积小、启动快、运行流畅。
- **🔒 数据隐私安全**：所有数据均在您本机处理，绝不上传云端，百分百保障学生与教学数据的安全。
- **📦 开箱即用**：提供完整安装包，无需复杂配置，下载安装即可投入使用。

## 📦 获取与开始使用

1. **下载安装**：前往 [Releases](https://github.com/你的用户名/OffWorkKit/releases)
   页面，下载适用于您操作系统（Windows/macOS/Linux）的最新安装包。
2. **安装运行**：像安装其他软件一样完成安装，双击即可开启效率之旅。
3. **系统要求**：最低要求Windows 10 1809 或更高版本，推荐使用64位系统。
